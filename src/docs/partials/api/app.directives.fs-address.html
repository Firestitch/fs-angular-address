fs-address
directive in module app

Usage

as element:
<fs-address
       fs-options="{object}"
       fs-address="{object}">
</fs-address>

Parameters

ParamTypeDetails
fs-optionsobject

Options to configure directive.

PropertyTypeDetails
countriesarray

List of all possible countries

disabledboolean

Sets the disabled attribute on the elements

mapboolean

Show or hide map

domesticsarray

An array of country codes that are to be placed at the top of the countries dropdown

addressobject

address field options

  • Required validation rule
  • Name used to map to the address model
address2object

address2 field options. If false - field not showing

  • Required validation rule
  • Name used to map to the address model
cityobject

city field options

  • Required validation rule
  • Name used to map to the address model
regionobject

region field options

  • Required validation rule
  • Name used to map to the address model
zipobject

zip field options

  • Required validation rule
  • Name used to map to the address model
countryobject

country field options

  • Required validation rule

Data for showing. Possible fields:
